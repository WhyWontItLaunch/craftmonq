# craftmonq - FFXI addon - Ashita v4
Modification of atom0s' craftmon addon (https://github.com/AshitaXI/Ashita-v4beta/tree/main/addons/craftmon). Adds result totals and message toggles.

## Text lines:
<img width="505" height="94" alt="image" src="https://github.com/user-attachments/assets/74f32888-e379-45c4-992e-f9e247bbf5e1" />

Result - Result of the craft attempt. Normal quality, break, or high-quality 1-3. **Shown** by default.\
Item - The item and quantity produced by the craft. **Shown** by default.\
Total - Lists total number of crafts and the count of each result. **Hidden** by default.

## Commands:
<img width="712" height="125" alt="image" src="https://github.com/user-attachments/assets/5b7dee83-1587-4147-b59f-24ed0bd7008d" />

Commands can be entered with /craftmanq or /cmq.\
Help - standard help command. Displays the addons help information.\
Show - can use used to toggle **on** any or all of the text lines. For example, '/craftmonq show total' to show the total.\
Hide - can use used to toggle **off** any or all of the text lines. For example, '/craftmonq hide result' to hide the result.\
Start - starts tracking results. Results are tracked by default.\
Stop -  stops tracking results. \
Total - prints the current total and count.\
Reset - resets the total and count.
